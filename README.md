# fomu vga console

![demo](https://raw.githubusercontent.com/hinzundcode/fomu-vga-console/master/demo.jpg)

## pinout

* user_1 to R/G/B (VGA pins 1, 2 and 3)
* user_2 to Ground
* user_3 to HSYNC (VGA pin 13) 
* user_4 to VSYNC (VGA pin 15)
